    Aufgabe 1 - Einkaufsliste (Link)

    In dieser Aufgabe geht es darum, die Applikationslogik einer bestehenden HTML-Seite zu erstellen. Dabei handelt es sich um eine Einkaufsliste, der über ein Input-Feld neue Einträge hinzugefügt werden können. Hinzugefügte Einträge wandern in die Einkaufsliste. Ein Klick auf einen der Artikel markiert diesen als erledigt (CSS-Klasse "ok"). Klickt man erneut auf den Artikel wird die Markierung wieder rückgängig gemacht. Über den "Liste leeren"-Button werden alle Einträge gelöscht.

    Regeln:

    • Der Code muss sich in der "app.js" befinden
    • jQuery ist eingebunden und darf benutzt werden
    • Die index.html darf nicht bearbeitet werden
    • Es dürfen keine weiteren Dateien erstellt werden
    • "Don't pollute the global namespace!"

    Entwicklung:

    Für diese Aufgabe benötigen wir keinen Webserver. Die "index.html" kann also direkt vom Filesystem im Browser geöffnet werden

    Themengebiete der Aufgabe:

    jQuery, DOM-Manipulation, IIFE, global namespace
